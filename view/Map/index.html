分类：地图 Map [-] [+]

游戏地图，例如 艾泽拉斯，东方王国，熔火之心等 从以下方法继承

方法

GetAreaId

返回指定X、Y和Z坐标处的Map 区域ID
GetDifficulty

返回 Map 的难度
GetHeight

返回 Map 指定X和Y坐标处的高度
GetInstanceData

返回 Map 的实例数据表, 如果存在
GetInstanceId

返回 Map 的实例ID
GetMapId

返回 Map 的ID
GetName

返回 Map 的名称
GetPlayerCount

返回当前在 Map 上的玩家人数，不包括GM.
GetPlayers

返回一个表，其中包含地图中的所有当前 Player
GetWorldObject

如果已重生，则通过其GUID从地图返回 WorldObject
IsArena

如果 Map 是竞技场 BattleGround 则返回true 否则返回false
IsBattleground

如果 Map 非竞技场 BattleGround 则返回true 否则返回false
IsDungeon

如果 true if the Map 是地下城副本 则返回true 否则返回false
IsEmpty

如果 Map 中没有 Player 则返回true 否则返回false
IsHeroic

如果 Map 难度为英雄 则返回true 否则返回false
IsRaid

如果 Map 为团队副本 则返回true 否则返回false
SaveInstanceData

将 Map 的实例数据保存到数据库
SetWeather

天气设置可有可无
